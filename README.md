# Transparent Desktop Calendar

Java (Swing) で作った、デスクトップに馴染む透明なカレンダーガジェットです。

## 特徴
- 背景が完全に透明なミニマルデザイン
- マウスドラッグで好きな場所に配置可能
- リアルタイムの時計表示（秒数まで）
- 2026年の日本の祝日に対応（赤色表示）
- 今日の日付をハイライト表示

## 使い方
1. このリポジトリにある 「CalendarSetup.exe」 をダウンロードします。
2. ダウンロードした 「CalendarSetup.exe」 を実行してインストールしてください。
3. インストール完了後、すぐにカレンダーが表示されます。次回のPC起動時からも自動で起動します。

## 開発環境
- Java 17以上
- VS Code (Extension Pack for Java 推奨)
